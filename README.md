# Privacy Policy for ShiftPay
最終更新日: 2026年3月18日

ShiftPay（以下「本アプリ」）は、勤務時間および収入の記録・表示を目的としたアプリです。
本プライバシーポリシーは、本アプリにおける利用者情報の取扱いについて説明するものです。

## 1. 取得する情報
本アプリは、以下の情報を取り扱います。

### 1-1. ユーザーが入力する情報
- 勤務開始時刻
- 勤務終了時刻
- 勤務時間
- 時給等の金額設定
- 履歴データ
- 初回設定やアプリ内設定に関する情報

### 1-2. 自動的に取得する情報
本アプリは、現時点で、氏名、メールアドレス、電話番号、位置情報、連絡先、写真、音声、広告 ID 等の個人情報を自動取得しません。

## 2. 情報の利用目的
本アプリは、取得した情報を以下の目的で利用します。
- 勤務時間の計測および表示
- 金額計算機能の提供
- 履歴の保存および表示
- 初回オンボーディングや設定状態の管理
- アプリの機能改善、不具合修正、および品質向上

## 3. 情報の保存場所
本アプリで入力または生成されたデータは、原則としてユーザーの端末内に保存されます。

開発者は、ユーザーの勤務履歴、金額設定、勤務状態その他の入力データを外部サーバーへ送信・保存しません。

## 4. 第三者提供
本アプリは、ユーザー情報を第三者へ販売、提供、共有しません。

ただし、法令に基づき開示が必要となる場合を除きます。

## 5. 外部サービス
本アプリは、現時点で広告配信サービスを利用していません。
また、現時点で本番の課金機能は未実装です。

将来、Google Play Billing 等の外部サービスを利用する場合には、本ポリシーを更新し、必要に応じて利用者へ通知します。

## 6. データ保持および削除
本アプリ内のデータは、ユーザーが削除するまで端末内に保持されます。

ユーザーは、以下の方法でデータを削除できます。
- アプリ内の履歴削除機能を利用する
- アプリのデータを端末設定から削除する
- アプリをアンインストールする

なお、端末内に保存されたデータは、アプリをアンインストールした場合または端末側でアプリデータを削除した場合に削除されることがあります。

## 7. セキュリティ
本アプリは、ユーザー情報の漏えい、滅失、毀損の防止その他安全管理のため、合理的な範囲で必要な措置を講じます。

ただし、端末そのもののセキュリティは、ユーザー自身の管理にも依存します。

## 8. 子どもの個人情報
本アプリは、子ども向けに特化したサービスではありません。
また、現時点で個人情報を収集する設計ではありません。

## 9. プライバシーポリシーの変更
本ポリシーの内容は、法令の改正やアプリ機能の変更等に応じて、予告なく変更されることがあります。
変更後のプライバシーポリシーは、公開後に効力を生じます。
